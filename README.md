# Blog Pessoal

Projeto desenvolvido como parte da minha formação em **FullStack JavaScript pela Generation Brasil**.

Este repositório registra a construção de uma API para um sistema de **Blog Pessoal**, criada com **NestJS**, com foco em prática de back-end, organização de código, banco de dados, CRUD, relacionamento entre entidades e implementação de segurança com autenticação.

## Status do projeto

🚧 **Em desenvolvimento**  
Todas as etapas deste projeto ainda estão em andamento, conforme o avanço das atividades propostas durante o bootcamp.

## Objetivo

Desenvolver uma aplicação back-end para um blog pessoal, colocando em prática conceitos fundamentais de desenvolvimento web, como:

- criação de API REST
- operações CRUD
- organização em camadas
- integração com banco de dados
- relacionamento entre entidades
- autenticação e segurança com token JWT

## Tecnologias utilizadas

- **Node.js**
- **TypeScript**
- **NestJS**
- **TypeORM**
- **MySQL**
- **JWT**
- **Bcrypt**
- **Insomnia**
- **Git e GitHub**

## Funcionalidades já trabalhadas

- cadastro de usuários
- atualização de usuários
- consulta de usuários
- autenticação de usuário
- proteção de rotas com JWT
- cadastro, consulta, atualização e exclusão de postagens
- cadastro, consulta, atualização e exclusão de temas
- relacionamento entre usuário, postagem e tema

## Estrutura do projeto

O projeto está organizado em recursos separados, seguindo uma estrutura modular:

- **Auth**
- **Usuário**
- **Tema**
- **Postagem**

## Aprendizados envolvidos

Durante a construção deste projeto, venho praticando conteúdos importantes da formação, como:

- desenvolvimento de API com NestJS
- uso de decorators
- modularização
- injeção de dependência
- persistência de dados com TypeORM
- relacionamentos entre tabelas
- autenticação e autorização
- testes de endpoints com Insomnia
